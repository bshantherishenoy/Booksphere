# 📚 Booksphere - DevOps Portfolio Project

## Overview

Booksphere is a full-stack book tracking application that allows users to manage and track books they have read. The project was built not only as an application but also as a hands-on DevOps learning platform covering containerization, CI/CD, cloud deployment, infrastructure automation, and production best practices.

---

## Features

### Application Features

* User authentication
* Book management
* Borrow record tracking
* PostgreSQL database integration
* Responsive HTML templates
* Health check endpoint
* Dockerized deployment

### Planned Features

* Wishlist management
* Reading progress tracking
* Book ratings and reviews
* Search and filtering
* User dashboard
* Redis caching
* AWS RDS integration
